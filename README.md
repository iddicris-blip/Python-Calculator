# Python Calculator

A simple yet functional Python calculator that performs basic arithmetic operations.

## Features

- **Basic Arithmetic Operations**: Addition, subtraction, multiplication, and division
- **User-Friendly Interface**: Easy-to-use command-line interface
- **Error Handling**: Graceful handling of invalid inputs and edge cases
- **Supports Multiple Operations**: Chain calculations or perform single operations

## Installation

### Prerequisites
- Python 3.6 or higher

### Setup

1. Clone the repository:
```bash
git clone https://github.com/iddicris-blip/Python-Calculator.git
cd Python-Calculator
```

2. Run the calculator:
```bash
python calculator.py
```

## Usage

### Basic Operations

The calculator supports the following operations:

- **Addition** (`+`): Add two numbers
- **Subtraction** (`-`): Subtract two numbers
- **Multiplication** (`*`): Multiply two numbers
- **Division** (`/`): Divide two numbers

### Example

```bash
$ python calculator.py
Enter first number: 10
Enter operation (+, -, *, /): +
Enter second number: 5
Result: 15
```
